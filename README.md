# rust_dev_intern_task3

*Rust Hash Finder Application.*

*It searches for numbers whose hashes end with N zeros, up to F results*

## How to build
```console
cargo build
```
or
```console
cargo build --release
```

## How to run
Examples:
```console
cargo run -- -F 6 -N 3
```
or
```console
cargo run --release -- -N 4 -F 7
```
